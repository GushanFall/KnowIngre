# 知食分子 KnowIngre

> 吃，是一门学问

轻量级智能厨房管理工具，支持 AI 生成菜谱、食材库存追踪、今日菜单规划。内置多用户系统和家庭共享。

## 功能

### 菜谱库
- AI 一键生成菜谱（输入菜名 → 自动填入食材和步骤）
- AI 微调修改（"换牛肉"、"减少用油"）
- 手动录入，支持标签分类、评分、搜索
- 20 种预设标签，AI 生成时自动选用，避免标签散乱
- 记录最近制作日期

### 食材库存
- 10 个分类（蔬菜、肉类、蛋奶、调料等）
- 智能匹配可制作的菜品，标注缺失食材
- 模糊匹配：「鸡胸肉」自动匹配库存里的「鸡胸肉」
- 做完菜后库存归零自动清理

### 今日菜单
- 采购清单：汇总所需食材，标注已有/缺货
- 做完自动扣减库存，标记制作日期
- 支持逐道确认或一键全部做完

### 家庭共享
- 个人/家庭双模式，数据完全隔离
- 邀请码加入，成员共同编辑菜谱和库存
- 单个菜谱或批量同步到对面范围

### 多用户
- 注册登录，bcrypt 加密
- 管理员可重置他人密码
- 数据 JSON 导入导出

## 本地运行

```bash
# 1. 安装 Bun
curl -fsSL https://bun.sh/install | bash

# 2. 启动
bun run server.ts

# 3. 打开
open http://localhost:3000
```

首次启动自动创建数据库。停止按 `Ctrl+C`，重启不会丢失数据。

## 使用流程

1. 注册账号（第一个用户自动成为管理员）
2. 设置中填入 API Key（默认 DeepSeek 接口）
3. 菜谱库新建菜谱，或输入菜名让 AI 生成
4. 库存中添加家中食材
5. 菜谱卡片点「加入菜单」，切换到今日菜单查看采购清单
6. 做完点「做完了」，库存自动扣减
